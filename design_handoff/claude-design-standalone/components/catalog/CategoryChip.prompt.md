Large filter chip for the catalog category toolbar; active state flips to the blue gradient.

```jsx
<CategoryChip label="Todos" icon={<Blocks size={16} />} count={128} active />
<CategoryChip label="Bebidas" icon={<GlassWater size={16} />} count={24} />
```

Use lucide-react icons. Lay out in a 3–4 column grid. Exactly one chip is `active` at a time.
