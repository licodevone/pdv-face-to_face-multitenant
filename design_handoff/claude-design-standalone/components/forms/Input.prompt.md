Rounded text field with a soft translucent fill and blue focus ring; pairs a muted label above the control.

```jsx
<Input label="Nome do cliente" placeholder="Ex.: Maria Souza" />
<Input label="CPF" hint="Apenas números" inputMode="numeric" />
```

Omit `label` to render a bare input. Use `hint` for helper text. Inherits the global input focus styles.
