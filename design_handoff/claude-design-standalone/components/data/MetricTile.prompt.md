Compact KPI tile for dashboard summary rows.

```jsx
<MetricTile label="Ticket médio" value="R$ 48,90" note="32 venda(s) no período" tone="success" />
<MetricTile label="Sessão do caixa" value="Caixa aberto" note="Desde 08:14" tone="accent" />
```

Tones: `neutral` / `success` / `accent`. Value renders large with tabular numerals.
